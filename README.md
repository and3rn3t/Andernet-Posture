# Project Repository

This project is set up to be version-controlled with Git and pushed to GitHub.

## 1) Initialize Git locally

Open Terminal at the project root (the folder containing your Xcode project/workspace) and run:

```bash
# Initialize repository
git init

# Ensure our ignore rules are in place
git add .gitignore

# Add all current files
git add -A

# First commit
git commit -m "Initial commit"
